# Contribuindo com o MeuRemedioEmDia

Obrigado por contribuir! 🎉

## Fluxo recomendado

1. Faça fork do projeto.
2. Crie uma branch descritiva (`feat/nome-da-feature`).
3. Faça commits pequenos e objetivos.
4. Abra PR descrevendo:
   - problema resolvido,
   - abordagem adotada,
   - impactos em acessibilidade/segurança,
   - como testar.

## Boas práticas

- Priorize clareza para público idoso (fonte legível, botões grandes, contraste alto).
- Evite textos ambíguos em temas de saúde e direitos.
- Nunca inclua dados pessoais reais em exemplos.

## Checklist de PR

- [ ] Código legível e consistente.
- [ ] Sem segredos no repositório.
- [ ] README/documentação atualizados quando necessário.
- [ ] Fluxos críticos revisados (medicação, urgência, contatos).
